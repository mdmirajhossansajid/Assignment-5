Difference between var, let, and const
var: Function-scoped, can be re-declared and updated, hoisted with undefined.
let: Block-scoped, can be updated but not re-declared in the same scope.
const: Block-scoped, cannot be updated or re-declared, must be initialized at declaration.

What is the spread operator (...)?

The spread operator (...) is used to expand elements of an array or object into individual elements.

Uses:

Copy arrays/objects
Merge arrays/objects
Pass multiple values in functions

Difference between map(), filter(), and forEach()
map(): Returns a new array by transforming each element.
filter(): Returns a new array with elements that satisfy a condition.
forEach(): Executes a function for each element, does not return a new array.

What is an arrow function?

An arrow function is a short syntax for writing functions using =>.
It does not have its own this and inherits it from the surrounding scope.

What are template literals?

Template literals are strings defined using backticks (`) that allow embedded expressions using ${} and support multi-line strings.
